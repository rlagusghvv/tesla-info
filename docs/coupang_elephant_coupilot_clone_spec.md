# 쿠팡 코끼리 — Coupilot.net 1차 동일본(클론) 스펙

> 기준 레퍼런스: https://www.coupilot.net/
> 작성: 2026-02-12
> 목표: **랜딩 1차 동일본** (정보구조/비주얼 리듬/섹션 구성/컴포넌트 동작을 최대한 유사하게)

## 전제/현재 상태
- 현재 `tesla-info/_repo`에는 쿠팡 코끼리 프로젝트 코드/폴더가 존재하지 않음.
- 따라서 이 문서는 **클론 구현에 필요한 요구사항/컴포넌트/레이아웃 스펙**을 정의하는 문서임.
- 구현 repo/프레임워크(Next.js/Remix/Vite 등)는 미확정 → 아래는 프레임워크 불문 공통 설계.

---

## 1) 페이지 정보구조 (IA)

### 1. Header / Top Nav (sticky)
- 좌측: 로고(COUPILOT)
- 메뉴: `요금제`, `품질`, `관리`, `컨설팅 수익(?)` 등(실제 메뉴명은 사이트 기준으로 맞추기)
- 중앙/우측: 
  - 검색 바/입력(placeholder: "오늘 뭐…" 류)
  - CTA 버튼: `로그인` 또는 `시작하기`

### 2. Hero Section (above-the-fold)
- 좌측:
  - 서브 배지: "쿠팡에서만…" 류 작은 pill
  - 메인 카피(대형):
    - "사업이 막막한 순간, 답은 데이터에 있습니다." 톤
  - 설명 문단(중간)
  - CTA 2개: Primary(검정 버튼), Secondary(텍스트/라인)
  - 소셜 프루프(아바타 + "10,000+" 등)
- 우측:
  - 대형 제품 목업 이미지/카드 스택
  - 작은 플로팅 카드(상승률 +300% 등)

### 3. Features Section (tabbed)
- 상단: 탭(라운드 pill): `상품 분석`, `소싱 분석`, `광고 분석` 등
- 탭별:
  - 좌측: 헤드라인 + 1~2줄 설명
  - 우측: 제품 카드(리포트/차트/지표)

### 4. Video/Proof Section (dark background)
- 배경: 다크(블랙/딥 네이비)
- 좌측: 헤드라인(화이트+강조 컬러)
- 우측: 영상 썸네일 카드 3개(가로 스크롤 or 그리드)

### 5. Metrics Section (light gradient)
- 큰 숫자/지표 2~3개(예: 0%, +0만원 등)
- 설명 텍스트

### 6. Testimonials/Chat Review Section
- 헤드라인: "성공한 대표님들의…" 류
- 카드 영역: 여러 리뷰 카드(초기에는 skeleton/placeholder 가능)

### 7. Footer
- 회사 로고 + 링크들 + 사업자 정보

---

## 2) 디자인 시스템(대략)
- 폰트: 산세리프(프리텐다드/Inter 계열) + 굵은 타이틀
- 컬러:
  - 배경: 화이트 + 핑크/보라 그라데이션 포인트
  - 강조: 핫핑크/마젠타 계열 + 블랙 CTA
- 버튼:
  - Primary: 검정 배경, 라운드(12~18), 화이트 텍스트
  - Secondary: 라운드 라인 or 텍스트
- 카드:
  - 라운드 16~22
  - subtle shadow + border(연한)

---

## 3) 인터랙션
- Header: 스크롤 시 sticky 유지
- Features 탭: 탭 전환 애니메이션(페이드/슬라이드)
- Hero 우측: 카드가 겹치며 살짝 떠 있는 느낌(absolute positioning)
- 영상 카드: hover 시 shadow/scale(모바일은 tap)

---

## 4) 구현 우선순위 (MVP 동일본)
1) 전체 섹션 구조/레이아웃 1:1 맞추기
2) 타이포/색감/여백/라운드/그림자까지 최대한 유사
3) 탭 동작 + 기본 애니메이션
4) 이미지/카피는 일단 placeholder로 채우되, 길이/위계는 동일

---

- 리뷰 카드용 아바타/스크린샷(placeholder)
