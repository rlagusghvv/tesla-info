# Coupang Elephant — 미완료 리스트 (10시 목표)

기준 URL: https://app.splui.com/app/

## ✅ 오늘 오전까지 완료된 것
- `app.splui.com`은 Cloudflare Tunnel로 **Next(3333)** 를 서빙.
- `/app/` 홈에서 **신규 디자인 + 레거시 콘솔(iframe)** 이식 완료.
- 신규 디자인 페이지 추가:
  - `/app/recommend` : 추천 리스트/상태 + 선택 업로드(직렬 enqueue)
  - `/app/upload` : URL preview + enqueue
  - `/app/console` : 레거시 콘솔 iframe
- 추천 다중 선택 업로드 시 누락되던 문제(버스트 요청) 방지:
  - sqlite busyTimeout + WAL 적용(동시 enqueue 시 SQLITE_BUSY 완화)

## ❗미완료(10시까지 해야 할 것)
1) **로그인/세션 UX 통합**
   - 지금은 레거시 콘솔에서 로그인(세션 저장)해야 추천/업로드 API가 정상 동작.
   - 신규 UI에서 로그인 상태 확인(`/api/me`) 및 로그인 버튼/가이드 제공 필요.

2) **신규 UI에서 기능 100% 이식(완전 동일)**
   - 레거시 콘솔의 모든 탭/기능(설정, 프리셋, 업로드 큐 실행/취소/재시도, 히스토리 등)을 신규 UI 컴포넌트로 “완전 이식”은 아직.
   - 현재는 iframe 임베드로 1차 이식(기능 제공) 상태.

3) **추천 품질/추천→업로드 자동화 고도화**
   - 추천이 ‘경쟁력 있는 상품’ 위주로 채워지도록 seed/필터/스코어링 개선 필요.
   - “사용자가 디깅 안 해도 되는” 수준까지 자동화(키워드/카테고리 전략) 미완.

4) **추천 배치 업로드 end-to-end 검증**
   - `추천 채우기 → 다중 선택 → 선택 업로드(직렬) → 큐 실행` 전체를 실제로 10개 단위로 돌려서
     실패 코드별 원인/가이드(이미지/레이트리밋/카테고리/옵션 등) 정리 필요.

## 10시까지의 ‘완료’ 정의(현실적)
- 대표님이 접속하는 메인 URL `https://app.splui.com/app/`에서:
  - 신규 디자인 유지
  - 레거시 기능이 iframe으로라도 **즉시 사용 가능(추천/업로드/큐 포함)**
  - 로그인 필요 시 안내/링크가 명확
  - 추천 다중 선택 업로드는 누락 없이 큐에 들어감
