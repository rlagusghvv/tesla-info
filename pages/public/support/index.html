개인정보처리방침 이용약관

지원

문의: support@splui.com
설치/설정 중 문제가 있으면 아래 체크리스트부터 확인해주세요.

1) 위치 권한(가장 중요)

  • iPhone 설정 → 개인정보 보호 및 보안 → 위치 서비스 → SubDash
  • 항상 허용
  • 정확한 위치 켜기

이 설정이 아니면 속도/좌표 레이턴시가 커져서 알림 타이밍이 나빠질 수 있습니다.

2) Tesla route ON 확인

  • 테슬라 차량에서 순정 내비로 목적지 설정
  • 앱 Car Mode 화면에서 Tesla route: ON 표시 확인

OFF면 Fleet 토큰 만료, 백엔드/터널 연결 이슈일 수 있습니다.

3) 카카오 경로 API 키

  • 카카오 REST API 키가 있어야 목적지 좌표로 경로를 만들어 “다음 카메라”를 계산할 수 있습니다.
  • 앱에서 Kakao key 필요가 뜨면 키가 비어있거나 저장이 안 된 상태입니다.

4) 서버/데이터 체크

  • 데이터셋: /api/data/speed_cameras_kr
  • 헬스체크: /health

5) 자주 묻는 질문

  • Q. 폰 화면을 켜야 하나요? A. 아니요. Car Mode 켜둔 채로 소리만 켜도 됩니다.
  • Q. 카플레이처럼 테슬라 화면에 띄워주나요? A. 아니요. 순정 내비를 메인으로 유지합니다.
