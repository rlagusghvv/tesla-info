개인정보처리방침

시행일: 2026-02-20
문의: support@splui.com

1. 처리하는 정보

  • 위치 정보: 주행 보조, 속도/카메라 알림 기능 제공을 위해 사용합니다.
  • Tesla 인증 정보(토큰): Tesla Fleet API 연동 기능 제공을 위해 사용합니다.
  • 앱 설정값: 사용자 옵션, API 설정 등 기능 유지를 위해 사용합니다.

2. 이용 목적

  • 차량 상태 조회 및 차량 제어 요청 처리
  • 주행 경로 기반 카메라/속도 경보 제공
  • 오류 분석 및 안정성 개선

3. 저장 위치와 보관 기간

  • 민감한 인증 정보는 기기 내 안전 저장소(Keychain)에 저장됩니다.
  • 사용자가 로그아웃하거나 앱을 제거하면 관련 정보는 삭제됩니다.
  • 운영 로그는 장애 대응 목적의 최소 범위로만 보관됩니다.

4. 제3자 서비스 연동

  • Tesla Fleet API
  • Kakao 지도/경로 API
  • 공공데이터(무인교통단속카메라)

5. 이용자 권리

  • iOS 설정에서 위치 권한을 언제든 변경할 수 있습니다.
  • 앱 내 계정 해제로 인증 토큰을 제거할 수 있습니다.
  • 개인정보 관련 문의는 support@splui.com 으로 접수할 수 있습니다.

6. 정책 변경 고지

본 방침은 법령 또는 서비스 변경에 따라 수정될 수 있으며, 변경 시 본 페이지를 통해 고지합니다.

TeslaSubDash는 기능 제공에 필요한 최소한의 정보만 처리합니다.
