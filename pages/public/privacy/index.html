개인정보처리방침

시행일: 2026-02-13
문의: support@splui.com

1. 수집하는 정보

Subdash는 주행 보조 기능 제공을 위해 아래 정보를 기기에서 처리할 수 있습니다.

  • 위치 정보: 속도/주행 좌표 계산 및 단속 카메라 경보를 위해 사용합니다.
  • Tesla 계정 인증 정보(토큰): Tesla Fleet API 접근을 위해 iOS Keychain에 저장될 수 있습니다.
  • 설정 값: 카카오 API 키, 앱 옵션 등(기기 내 저장).

2. 이용 목적

  • 테슬라 차량 상태/명령 및 경로 동기화(사용자 동의 기반)
  • 단속 카메라 접근/과속 경보 제공
  • 앱 연결 상태/오류 진단(최소 범위)

3. 보관 및 파기

  • 민감한 인증정보는 기기(Keychain)에 저장되며, 사용자가 로그아웃/삭제 시 제거됩니다.
  • 서비스 품질 개선을 위한 로그에는 개인식별정보를 저장하지 않도록 설계합니다.

4. 제3자 제공 및 처리위탁

  • Tesla Fleet API, 지도/경로 API 등 외부 서비스와 통신할 수 있습니다(기능 제공 목적).
  • 결제는 App Store 인앱 결제를 통해 처리됩니다(Apple).

5. 이용자 권리

  • iOS 설정에서 위치 권한을 변경할 수 있습니다.
  • 앱 내 로그아웃으로 토큰을 삭제할 수 있습니다.

본 방침은 제품/정책 변경에 따라 업데이트될 수 있습니다. 변경 시 이 페이지에 반영합니다.
