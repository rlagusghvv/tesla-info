# Coding Rules

## Frontend Skills Auto-Apply
- UI/프론트 작업(웹/Flutter)에서는 `frontend_skills/contracts/DESIGN_CONTRACT.md`를 항상 준수한다.
- 구현 후 반드시 `frontend_skills/prompts/UI_POLISH_PASS.md` + `frontend_skills/prompts/STATE_COVERAGE_PASS.md`를 수행한다.
- 웹 작업이면 `frontend_skills/prompts/WEB_NEXTJS_SHADCN_PASS.md`를 추가 적용한다.
- Flutter 작업이면 `frontend_skills/prompts/FLUTTER_UI_PASS.md`를 추가 적용한다.
- 목적은 폴리시(UI/UX 품질 개선)이며 기능/비즈니스 로직 변경은 금지한다.
